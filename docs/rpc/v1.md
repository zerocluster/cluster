# RPC v1

### API connect URLs

- API HTTP URL: `http://cluster/api/`
- API WebSockets URL: `ws://cluster/api/`

<!-- tabs:start -->

#### **WebSockets**

```javascript
import Api from "@softvisio/core/api";

const api = new Api( "ws://cluster/api/" );
```

#### **HTTP**

```javascript
import Api from "@softvisio/core/api";

const api = new Api( "http://cluster/api/" );
```

<!-- tabs:end -->

## Cluster events

### Publish event

<!-- tabs:start -->

#### **JavaScript**

```javascript
const res = await api.call( "/v1/events/publish", cluster_id, event );
```

#### **Shell**

```shell
curl \
    -H "Authorization: Bearer <YOUR-API-TOKEN>" \
    "http://cluster/api/v1/events/publish"
```

<!-- tabs:end -->

- `cluster_id` {string}

<details>
    <summary>JSON schema</summary>

```json
{
    "type": "string"
}
```

</details>

- `event` {Object}

<details>
    <summary>JSON schema</summary>

```json
{
    "type": "object",
    "properties": {
        "name": {
            "type": "string"
        },
        "users": {
            "type": ["null", "string", "array"]
        },
        "data": {
            "type": "string"
        }
    },
    "additionalProperties": false,
    "required": ["name"]
}
```

</details>
